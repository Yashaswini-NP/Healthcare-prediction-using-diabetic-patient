# Diabetes Prediction & Analysis using Machine Learning and Power BI

## 📌 Project Overview
This project focuses on predicting diabetes using machine learning and analyzing healthcare data using Power BI to generate meaningful insights.

The project was completed during my internship with Edunet Foundation (Dec 2023 – Jan 2024).

---

## 📊 Dataset
The dataset used is the Pima Indians Diabetes dataset containing 768 patient records with features like:
- Pregnancies
- Glucose
- Blood Pressure
- BMI
- Insulin
- Age
- Diabetes Pedigree Function

Target Variable:
- Outcome (0 = Non-diabetic, 1 = Diabetic)

---

## 🧹 Data Preprocessing
- Replaced invalid zero values in medical features with missing values
- Handled missing data using median imputation
- Scaled features for better model performance

---

## 🤖 Machine Learning Model
- Model Used: Logistic Regression
- Accuracy Achieved: ~78%
- Applied train-test split (80-20)

---

## 📈 Power BI Dashboard
An interactive dashboard was created to visualize key insights:

### Key Visualizations:
- KPI Cards (Total Patients, Diabetes %)
- Age vs Outcome
- Glucose Distribution (Bins)
- BMI vs Outcome
- Outcome Distribution (Pie Chart)
- Pregnancies Impact
- Insulin Analysis
- Genetic Risk (Diabetes Pedigree Function)

---

## 🔍 Key Insights
- Glucose is the strongest predictor of diabetes
- BMI and Age significantly influence diabetes risk
- Higher pregnancies increase likelihood of diabetes
- Genetic factors also contribute to diabetes
